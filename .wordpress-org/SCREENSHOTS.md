# WordPress.org listing assets

Drop the listing assets here. These are **not** shipped in the plugin zip (excluded via `.distignore`) but they:

1. Power the **WordPress.org** plugin page (via the deploy/SVN `assets/` folder), and
2. Render in the GitHub **README** "Screenshots" section.

## Needed files (PNG)

| File | Shown as | Suggested shot |
|---|---|---|
| `screenshot-1.png` | Screenshot 1 | The StackPress dashboard — all 185 tools with the RAM/JS/DB cost chips |
| `screenshot-2.png` | Screenshot 2 | A category (e.g. WooCommerce) with enable toggles + a conflict warning |
| `screenshot-3.png` | Screenshot 3 | A tool's settings modal open |
| `screenshot-4.png` | Screenshot 4 | Recommended setup / Agency Mode |

Recommended width **1280px** (matching the live dashboard). The captions on the
WordPress.org page come from the `== Screenshots ==` list in `readme.txt`.

## Branding (already here)

- `banner-1544x500.svg`, `banner-772x250.svg` — **export to PNG before uploading to .org** (banners can't be SVG)
